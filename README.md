Provides a convenience <i>CIColor</i> method for set up a color from a HEX string in <i>Swift</i>.

# Installation

<b>CocoaPods:</b>
<pre>
pod 'CIColorRGBA'
</pre>

<b>Manual:</b>
<pre>
Copy <i>CIColorRGBA.swift</i> to your project.
</pre>

# Using

<pre>
let color = CIColor(rgba: '#134E5E')
</pre>
